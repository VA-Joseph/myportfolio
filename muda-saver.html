Muda Saver

Overview: A USSD-based platform for users to track and place real-time food orders from vendors. Smartphone users get location-based tracking for faster delivery and efficiency.

Key Features:

  • Real-time food order tracking (USSD + smartphone)
  • Location-based vendor search
  • Vendor and customer management system
  • Optimizes time for both customers and vendors

Technologies Used:

  • USSD Gateway Integration
  • Mobile-Friendly Web Interface
  • Real-time GPS Tracking (for smartphones)
  • Backend Order Management System
